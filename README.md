# PlanBot - Telegram Task Planning Bot 📋

Умный телеграм-бот для планирования задач с автоматическим распределением по дням на основе нагрузки, приоритетов и дедлайнов.

## 🎯 Возможности

- **Умное гибридное планирование**: Автоматическое распределение задач по дням с учётом:
  - Времени, необходимого на выполнение
  - Приоритетов задач
  - Жёстких дедлайнов (с использованием обратного планирования)
  - Вашей дневной нагрузки
  
- **Интеграция с Google Calendar**: Автоматическая выгрузка вашего расписания в календарь.
  
- **Поддержка часовых поясов**: Корректная работа планировщика и уведомлений в вашем локальном времени.

- **Гибкие настройки**:
  - Настраиваемое количество рабочих часов в день
  - Выбор рабочих дней недели
  - Разделение задач на несколько дней при необходимости

- **Удобный интерфейс**:
  - Простые команды для управления задачами
  - Просмотр расписания на день/неделю
  - Отслеживание статуса задач

## 🧠 Алгоритм планирования

Реализован **гибридный алгоритм с учётом дедлайнов**:

1. **Сортировка задач**:
   - Сначала задачи с дедлайнами (по близости срока)
   - Затем по приоритету (10 = максимальный)
   - При равном приоритете - более короткие задачи

2. **Распределение**:
   - **Backward Planning (для дедлайнов)**: Бот планирует задачи как можно ближе к дедлайну, оставляя свободное место в ближайшие дни для текучки.
   - **Forward Planning (без дедлайнов)**: Задачи без жестких сроков планируются в ближайшие свободные окна.
   - Разбивает большие задачи на несколько дней.
   - Учитывает рабочие/нерабочие дни.

3. **Оптимизация**:
   - Не перегружает отдельные дни выше установленного лимита.
   - Автоматическая перепланировка при добавлении новых задач.

Подробное описание алгоритма читайте в [ALGORITHM_EXPLANATION.md](ALGORITHM_EXPLANATION.md).

- Go 1.21 или выше

- Google Cloud Project (для интеграции с календарем)

## 🚀 Установка и запуск

### Вариант 1: Docker (Рекомендуется) 🐳

```bash
# Клонируйте репозиторий
git clone <your-repo-url>
cd PlanBot

# Скопируйте пример конфигурации
cp env.example .env

# Отредактируйте .env и добавьте токен бота и настройки БД
nano .env

# Запустите через Docker Compose
docker-compose up -d

# Просмотр логов
docker-compose logs -f bot
```

### Вариант 2: Локальная установка

1. **Клонируйте репозиторий**:
```bash
git clone <your-repo-url>
cd PlanBot
```

2. **Создайте файл .env**:
```bash
cp env.example .env
# Отредактируйте .env
```

3. **Используйте Makefile для быстрой настройки**:
```bash
# Установить зависимости
make install

# Создать базу данных
make db-setup

# Запустить бота
make run
```

## 📱 Команды бота

### Основные команды:

- `/start` - Начать работу с ботом
- `/help` - Справка по командам

### Управление задачами:

- `/addtask` - Добавить новую задачу
  ```
  Формат: /addtask Название | часы | приоритет | дедлайн
  Минимум: /addtask Задача | 2
  Примеры:
  /addtask Написать отчёт | 4 | 8 | 25.12.2025
  /addtask Прочитать статью | 1.5 | 3
  ```

- `/mytasks` - Показать все задачи
- `/complete <ID>` - Отметить задачу как выполненную
- `/delete <ID>` - Удалить задачу

### Планирование:

- `/schedule` - Автоматически распределить задачи по дням (и экспорт в Google Calendar)
- `/today` - Показать расписание на сегодня
- `/week` - Показать расписание на неделю

### Настройки:

- `/settings` - Показать текущие настройки
- `/settings <часы> | <дни> | <время>` - Изменить настройки
  ```
  Пример: /settings 6 | 1,2,3,4,5 | 09:00-18:00
  ```
- `/timezone <имя>` - Установить часовой пояс (например, `Europe/Moscow`)

### Google Calendar:
- `/google_connect` - Подключить календарь
- `/google_status` - Проверить статус подключения
